CRONU Help Documentation

We are professional team to develop Wordpress theme and plugin.

Our team have project manager and web designer, Wordpress developer and front-end developer

CRONU is out first theme and wordpress theme for Blog and Eshop. Also, this theme include Buddypress, BBpress, etc.

1. Introduction of CRONU

CRONU Theme 1.0.1 is HTML & WordPress theme for Landing Page Site & Blog & Shop & Social Site & Forums. It’s perfect for web agency business, corporate business, personal and business portfolio, photography sites, and freelancer, services, blog and shop, social networking & forums.It's built on BootStrap with parallax support, is responsive, clean, modern, flat and minimal. CRONU Theme is e-commerce (WooCommerce) Compatible, WPML, RTL, Retina-Ready, SEO Friendly and with parallax, full-screen image is one of the best business themes.

2. Basic Setup
2.1. Install Theme

To start you must have a working version of WordPress. Almost all major web hosts provide one-click WordPress install and you can consult your host's help site for more information in this regard. For the manual install, you can consult WordPress Codex Installing WordPress.

Once you have a working version of WordPress, You need to download "All Files and Documentation" from ThemeForest download page and extract the downloaded zip to get various files including cronu.zip.

Install Theme Via WordPress Dashboard ( Recommended Way )

Go to 'Appearance > Themes' section

1. Click 'Add New' and select the 'Upload Theme' option
   

 

2. Click 'Upload Theme'
    

 

3. Choose the realhomes.zip file and press 'Install Now'
    

 

4. Once the theme is uploaded you need to activate it.
    

 

5. After activation go to "Appearance > Theme Options" to play with theme's settings.
    You also need to install some missing plug-ins for CRONU theme.
    

    if you face any problem during upload through the dashboard, please upload the theme using FTP as guided below.

Install Theme Via FTP ( Alternative Way )

  1. Access your hosting server using an FTP client like FileZilla
  2. Go to the 'wp-content/themes' folder of your WordPress installation
  3. Extract the cronu.zip file and put the cronu folder in 'wp-content/themes/' folder
  4. Go to 'WordPress Dashboard > Appearance > Themes' section to activate the theme
  5. After activation go to "Appearance > Customize" to play with theme's settings.

5.3. Layouts

What is Layout ?

Layout is the set of theme options.
Wordpress theme can has a number of layouts and only one can be activiated and affected to the website.
Each layout has its own a set of Theme Options, changing activation of layouts you can apply a different set of Theme Options.
You can change the display of theme by changing the active layout at a time.


Go to Appearance > Theme Options, you can config Theme Options for each layout.

 

To add a new layout enter a unique lowercase alphanumeric string (dashes allowed) in the text field and click "Save Layouts".

As well, you can activate, remove, and drag & drop the order; all situations require you to click "Save Layouts" for the changes to be applied.

When you create a new layout it will become active and any changes made to the Theme Options will be applied to it.
If you switch back to a different layout immediately after creating a new layout that new layout will have a snapshot of the current Theme Options data attached to it.
